# 数组

### 数组的操作

数组的操作大概有**查找、插入、删除**和**排序**

关于**查找**，常见的算法有**顺序查找**、**二分查找，**具体可以进入**「算法技巧系列 - 检索算法」**

关于**排序**，常见的基本排序算法有**冒泡排序**、**选择排序**、**插入排序**，高级排序算法有**希尔排序**、**归并排序**、**快速排序，**具体可以进入「**算法技巧系列 - 排序算法」**

### **数组中常见的两类算法**

**双指针**和**滑动窗口**是数组中常用的两种算法**。**具体可以进入**「算法技巧系列 - 双指针」**和 **「算法技巧系列 - 滑动窗口系列」**
